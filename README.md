# Truth-Or-Dare-Game-Java
Console-based Truth or Dare game in Java with random question generation using ArrayList and basic game logic.
# 🎭 Truth or Dare Game (Java Console)

## 📌 Description
A fun console-based Truth or Dare game developed using Core Java.  
The game randomly generates Truth or Dare challenges for players.

## 🎮 Features
- Random Truth or Dare selection
- Multiple questions using ArrayList
- Interactive console gameplay
- Replay option

## 🛠 Concepts Used
✔ ArrayList  
✔ Random Class  
✔ Loops  
✔ Conditional Statements  
✔ Scanner Input  

## 🎯 Game Flow
1. Player chooses Truth or Dare
2. System randomly selects a question
3. Player performs or answers
4. Repeat until exit

## ▶ How to Run
1. Open project in Eclipse / IntelliJ
2. Run main class
